# 石田堤・旧利根川水系の重ね合わせ地図（GitHub Pages）

更新日: 2026-02-24

このリポジトリは、ブラウザで開ける「重ね合わせ地図（HTML）」を GitHub Pages で公開するためのものです。

ファイル:
- index.html : 最新版（トップで開く）
- v4.html    : 保管用（index.html と同内容）

開き方:
- 公開URL（GitHub Pages）にアクセスすると index.html が表示されます
- v4.html は「/v4.html」で直接開けます

注意:
- 地図タイルや OSM のデータをブラウザから読み込みます（ネット接続が必要）
- Overpass 側が混雑して線が出ない場合は、ページ内の「再読み込み」を試してください


# Ishida Tsutsumi overlay map (GitHub Pages)

Updated: 2026-02-24

This repository publishes an interactive overlay map (HTML) via GitHub Pages.

Files:
- index.html : latest (opens by default)
- v4.html    : archived copy (same content as index.html)

How to open:
- Open the GitHub Pages URL to view index.html
- Open "/v4.html" to view the archived version

Notes:
- The page loads map tiles and OSM data in the browser (network access required)
- If lines do not appear, the Overpass server may be busy; try the "reload" button in the page
